## Breaking Changes
- client: Change `RetryIfFunc` from `func(*protocol.Request) bool` to `func(*protocol.Request, *protocol.Response, error) bool` (#239)
- client: Remove `WithMaxIdempotentCallAttempts` option, use `WithRetryConfig` instead (#239)
- http1: Remove `ClientOptions.MaxIdempotentCallAttempts`, use `ClientOptions.RetryConfig` instead (#239)

## Deprecations
- client: Deprecate `Client.SetRetryIf`, use `Client.SetRetryIfFunc` instead (#318)

## Features
- client/retry: Add `Config` with configurable delay policies (`DefaultDelayPolicy`, `FixedDelayPolicy`, `BackOffDelayPolicy`, `RandomDelayPolicy`, `CombineDelay`) (#239)
- client: Add `WithRetryConfig` option and `Client.SetRetryIfFunc` (#239)
- client: Add `DefaultRetryIf` for idempotent retry logic (#239)
- protocol: Add `RequestHeader.IsDelete`, `RequestHeader.IsOptions`, `RequestHeader.IsTrace` (#239)
- app: Add `RequestContext.IndentedJSON` for pretty-printed JSON responses (#289)
- json: Add `pkg/common/json` package with build-tag-based JSON implementation selection (#278)
- errors: Add `Newf`, `NewPublicf`, `NewPrivatef` (#258)
- recovery: Add `WithRecoveryHandler` option for custom panic handler (#304)
- network/standard: Add `Conn.CloseNoResetBuffer` (#265)
- hlog: Add `SystemLogger`, `SetSystemLogger` for framework-internal logging (#266)

## Fixes
- client/retry: Fix `RandomDelayPolicy` panic when `Config.MaxJitter` <= 0 (#282)
- client/retry: Fix `BackOffDelayPolicy` panic when `Config.Delay` <= 0 (#282)
- client/retry: Fix `Delay` panic when `Config.DelayPolicy` is nil (#282)
