## Features
- sse: Add `pkg/protocol/sse` package implementing Server-Sent Events per HTML spec (#1335)
  Includes `Event`, `Reader`, `Writer`, `NewWriter`, `AddAcceptMIME`, `GetLastEventID`, `SetLastEventID`.
- adaptor: Add `HertzHandler` to convert `http.Handler` to Hertz handler (#1327)
  Supports chunked responses, `http.Flusher`, `http.Hijacker`; deprecates `GetCompatRequest`/`GetCompatResponseWriter`.
- server: Add request race detection warning in HTTP/1 server (#1341)
- server: Add `SenseClientDisconnection` support for standard (non-netpoll) transport (#1322)

## Fixes
- protocol: Fix `Request.BodyStream`/`Response.BodyStream` side-effect mutation when returning `NoBody`/`NoResponseBody` (#1333)
- client: Fix SSE responses blocking until timeout; auto-enable stream mode for `text/event-stream` (#1329)
- server: Fix `Shutdown` not respecting `config.Options.ExitWaitTimeout`; move timeout logic into `route.Engine.Shutdown` (#1332)
- binding: Fix custom validator overridden by `BindConfig.Validator` (#1316)
- json: Restrict sonic/netpoll to amd64/arm64 on linux/darwin only (#1340)
