## Breaking Changes
- network: Add `SetWriteTimeout(t time.Duration) error` to `Conn` interface (#344)

## Deprecations
- route: Deprecate `GetTransporterName`, use `Engine.GetTransporterName` instead (#356)

## Features
- server: Add `WithWriteTimeout` option (#344)
- client: Add `WithWriteTimeout` option (#344)
- config: Add per-request `WithDialTimeout`, `WithReadTimeout`, `WithWriteTimeout` options (#344)
- errors: Add `ErrReadTimeout`, `ErrWriteTimeout`, `ErrDialTimeout` (#344)
- protocol: Add `RequestHeader.SetNoDefaultContentType` to suppress default Content-Type (#367)
- app: Add `RequestContext.GetInt32`, `RequestContext.GetUint`, `RequestContext.GetUint32`, `RequestContext.GetUint64`, `RequestContext.GetFloat32` (#375)
- route: Add `Engine.GetTransporterName` for per-engine transporter name (#356)
- config: Add `Options.TransporterNewer` for per-engine transporter configuration (#356)
