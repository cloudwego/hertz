## Breaking Changes
- binding: Rewrite binding package with new `Binder`/`StructValidator` interfaces; remove old global config functions (`SetLooseZeroMode`, `SetErrorFactory`, `MustRegTypeUnmarshal`, `MustRegValidateFunc`, `UseStdJSONUnmarshaler`, `UseGJSONUnmarshaler`, `UseThirdPartyJSONUnmarshaler`) (#541, #955)
- app: Change `ClientIPOptions.TrustedProxies map[string]bool` -> `ClientIPOptions.TrustedCIDRs []*net.IPNet` (#934)

## Features
- binding: Add `Binder` interface with `Bind`, `BindAndValidate`, `BindQuery`, `BindHeader`, `BindPath`, `BindForm`, `BindJSON`, `BindProtobuf` methods (#541)
- binding: Add `BindConfig`, `ValidateConfig` for configurable binding and validation (#541, #955)
- binding: Add `NewDefaultBinder`, `NewValidator` constructors and `DefaultBinder`, `DefaultValidator` getters (#541, #955)
- app: Add `RequestContext.BindQuery`, `RequestContext.BindHeader`, `RequestContext.BindPath`, `RequestContext.BindForm`, `RequestContext.BindJSON`, `RequestContext.BindProtobuf`, `RequestContext.BindByContentType` (#541)
- app: Add `RequestContext.SetBinder`, `RequestContext.SetValidator` (#541)
- server: Add `WithBindConfig`, `WithValidateConfig`, `WithCustomBinder`, `WithCustomValidator` options (#541, #955)
- server: Add `WithDisableHeaderNamesNormalizing` option (#940)
- client: Add `WithHostClientConfigHook` option for post-creation host client configuration (#938)
- route: Add `Engine.MarkAsRunning` to externally mark engine as running (#889)
- protocol: Add `RequestHeader.Del` convenience method (#950)
- protocol: Add `NoResponseBody` var for no-content stream detection (#936)

## Fixes
- hlog: Fix escaped character error printing in format-less log calls (#819)
- http1: Fix connection leak in stream mode when response has no content (204) (#936)

## Improvements
- app: Support CIDR ranges in `ClientIPOptions.TrustedCIDRs` for trusted proxy matching (#934)
