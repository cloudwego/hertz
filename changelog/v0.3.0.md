## Breaking Changes
- client: Remove `Client.SetDialFunc`, `WithDialFunc` option, and `client.DialFunc` type (#175)
- client: Change `ClientOptions.Dial func(addr string) (network.Conn, error)` -> `ClientOptions.Dialer network.Dialer` (#175)
- http1: Change `ClientOptions.Dial client.DialFunc` -> `ClientOptions.Dialer network.Dialer` (#175)
- http1/proxy: Change `SetupProxy(conn, addr, proxyURI, tlsConfig, isTLS)` -> `SetupProxy(conn, addr, proxyURI, tlsConfig, isTLS, dialer)` (#175)

## Deprecations
- dialer: Deprecate `SetDialer`, use `client.WithDialer` instead (#175)

## Features
- server: Add service registration with `WithRegistry` option (74dde60)
- discovery: Add `Resolver`, `Instance`, `Result`, `NewInstance` for service discovery (74dde60)
- registry: Add `Registry`, `Info` for service registration (74dde60)
- loadbalance: Add `Loadbalancer` interface and `NewWeightedBalancer` for weighted random load balancing (74dde60)
- sd: Add `Discovery` client middleware for service discovery integration (74dde60)
- config: Add `RequestOptions.Tags` to expose request tags (74dde60)
- dialer: Add `DefaultDialer` to retrieve global default dialer (#175)
